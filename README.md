# Student Grade Calculator (C++)

This project is a simple C++ program that calculates a student's total marks, percentage, and grade based on the scores entered for multiple subjects.

## Features
- Input marks for any number of subjects
- Calculates total marks
- Calculates percentage
- Assigns grade based on percentage
- Simple and beginner friendly

## Grading Criteria
- A: 90% and above
- B: 80% - 89%
- C: 70% - 79%
- D: 60% - 69%
- F: Below 60%

## How to Run
g++ main.cpp -o grade
./grade
